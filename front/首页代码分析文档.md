## 一、项目概述

### 1.1 项目定位

**基于零知识证明（ZKP）与轻量级规则的健康隐私系统** - 一个展示零知识证明技术在医疗健康数据隐私保护领域应用的前端演示系统。

### 1.2 核心价值主张

| 价值点 | 描述 |
|--------|------|
| **隐私第一** | 原始健康数据永不离开用户设备 |
| **可验证性** | 通过数学证明验证数据真实性 |
| **合规审计** | 满足医疗健康数据保护法规要求 |

### 1.3 核心功能模块

1. **HeroSection** - 首屏展示区域，粒子背景与滚动视差
2. **TrustSection** - 信任背书展示，合作伙伴轮播
3. **WorkflowSection** - 工作流程可视化，三步验证流程
4. **PrivacyDemo** - 隐私保护演示，ZKP与传统模式对比

```typescript
  endShrinkAt: 600,      // 完全收缩位置

// 计算滚动进度（0-1）
function calculateScrollProgress(currentScroll: number, config: ScrollConfig): number {
  const progress = (currentScroll - config.startShrinkAt) / 
                   (config.endShrinkAt - config.startShrinkAt);
  return Math.max(0, Math.min(1, progress));
}
```

**2. 高度动态调整**

**3. 滚动监听优化**

```typescript
useEffect(() => {

    if (requestRef.current) {
```

### 3.7 主题提供者 (ThemeProvider.tsx)

**文件路径**: `src/shared/utils/ThemeProvider.tsx`

#### 功能描述

提供全局主题状态管理和切换功能。

#### Context接口

```typescript
interface ThemeContextType {
  theme: Theme;           // 'day' | 'night'
  toggleTheme: () => void;
  isNight: boolean;
}
```

#### 初始化逻辑

```typescript
const [theme, setTheme] = useState<Theme>(() => {
  // 仅在客户端执行
  if (typeof window !== 'undefined') {
    // 1. 优先读取localStorage
    const savedTheme = localStorage.getItem('theme') as Theme | null;
    if (savedTheme) return savedTheme;
    
    // 2. 根据系统时间自动判断
    const hour = new Date().getHours();
    const isNight = hour < 6 || hour > 20;
    return isNight ? 'night' : 'day';
  }
  
  // 服务端默认值
  return 'night';
});
```

#### 主题切换

```typescript
const toggleTheme = () => {
  const newTheme = theme === 'day' ? 'night' : 'day';
  setTheme(newTheme);
  localStorage.setItem('theme', newTheme);
};
```

#### 关键技术点

- **SSR兼容**: 使用初始化函数避免水合错误
- **持久化**: localStorage保存用户偏好
- **自动判断**: 基于时间自动选择主题
